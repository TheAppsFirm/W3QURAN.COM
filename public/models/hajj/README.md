# Hajj 3D Models

Download these free models from Sketchfab and place them in this folder:

## Required Models

### 1. Masjid al-Haram (For Tawaf)
- **Download:** https://sketchfab.com/3d-models/masjid-al-haram-great-mosque-of-mecca-329bb58e51c94ea28382053a8d11d8d9
- **Filename:** `masjid-al-haram.glb`
- **How to download:**
  1. Visit the link above
  2. Click "Download 3D Model"
  3. Select "glTF" format
  4. Extract and rename to `masjid-al-haram.glb`
  5. Place in this folder

### 2. Kaaba (Alternative/Backup)
- **Download:** https://sketchfab.com/3d-models/kaaba-in-mecca-43041d42a0ae4cb58e20a86edc572688
- **Filename:** `kaaba.glb`
